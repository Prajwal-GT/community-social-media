# Community Social Media

A full-stack MERN social media web application.

## Features
- User Registration
- User Login (JWT Authentication)
- Create Posts
- Like Posts
- Dislike Posts
- Comment on Posts
- MongoDB Database

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure

community-social-media/
├── frontend/
└── backend/

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
